# 02. AI Is Not One File

When beginners think of AI,
they imagine one big magical file.

That is not how real systems work.

Real AI systems are modular.

They are built from parts.

Instead of:

ai.py  (5000 lines)

We build:

- brain.py
- memory.py
- tools.py
- voice.py
- vision.py
- ui.py

Each file has one responsibility.

This makes systems:
- easier to understand
- easier to debug
- easier to upgrade
